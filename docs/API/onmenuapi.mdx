# GUI Api 

## Make Functions

These functions are used to create GUI elements or menus, they should only be called once.

:::warning **Warning:** Read this or you might create bugs
some basic info that you require to follow otherwise you will make bugs.
-When making a menu, and you wanna add a toggle in it you give it the exact same MenuName & SubMenu string.
-Make Sure each Toggle,Button,Slider,ListBox has a UNIQUE name
:::


### Faith.MakeMenu

- This function will return false if it didn't work or true if it worked. If it fails your windowName is probably not unique!

```jsx title="MakeMenu Example"
-- Faith.MakeMenu(Icon icon, <string> MenuName,  <string> SubMenu)
Faith.MakeMenu(ICON_FA_USER, "veh", "Spawner")
```

### Faith.MakeToggle

- This function will return false if it didn't work or true if it worked. If it fails your windowName is probably not unique!


```jsx title="MakeToggle Example"
-- <bool> Faith.MakeToggle(<string> name, <string> MenuName, <string> SubMenu)
Faith.MakeToggle("Spawn Vehicle", "veh", "Spawner")
```

### Faith.MakeButton

- This function will return false if it didn't work or true if it worked. If it fails your windowName is probably not unique!


```jsx title="MakeButton Example"
-- <bool> Faith.MakeButton(<string> name, <string> MenuName, <string> SubMenu)
Faith.MakeButton("Spawn Vehicle", "veh", "Spawner")
```

### Faith.MakeSlider

- This function will return false if it didn't work or true if it worked. If it fails your windowName is probably not unique!

```jsx title="MakeSlider Example"
-- <bool> Faith.MakeSlider(<string> name, <int> min, <int> max, <string> MenuName, <string> SubMenu)
Faith.MakeSlider("Spawn Vehicle", 0, 100 "veh", "Spawner")
```

### Faith.MakeListBox

- This function will return false if it didn't work or true if it worked. If it fails your MenuName is probably not unique!

```jsx title="MakeListBox Example"
-- <bool> Faith.MakeListBox(<string> name, <string> MenuName, <string> SubMenu, <string> items...)
Faith.MakeListBox("Spawn Vehicle", "veh", "Spawner", "Hey", "Hi", "Adder", "Antigrind")
```

## Fetch Functions

These functions are used to fetch the value of the GUI elements you created before, you will probably want to call these inside a Worker.

:::warning **Warning:** Read this or you might create bugs
some basic info that you require to follow otherwise you will make bugs.
- Name of the fetches function, always has to be the same as the Made toggle.
- It must be from the same MenuName & Submenu you created it in.
:::

### Faith.FetchToggle

This function will return the state of the toggle, true if it's toggled false if it's not.

```jsx title="FetchToggle Example"
-- <bool> Faith.FetchToggle(<string> name, <string> MenuName, <string> SubMenu)
Faith.FetchToggle("Spawn Vehicle", "veh", "Spawner")
```


### Faith.FetchButton

This function will return the state of the button, true if it has been clicked before, false if it hasn't. After calling this function the state will be cleared and will return false until it's clicked again.

```jsx title="FetchButton Example"
-- <bool> Faith.FetchButton(<string> name, <string> MenuName, <string> SubMenu)
Faith.FetchButton("Spawn Vehicle", "veh", "Spawner")
```

### Faith.FetchSlider

This function will return the value of the slider.

```jsx title="FetchSlider Example"
-- <int> Faith.FetchSlider(<string> name, <string> MenuName, <string> SubMenu)
Faith.FetchSlider("Spawn Vehicle", "veh", "Spawner")
```

### Faith.FetchListBox

This function will return the currently selected item's string.

```jsx title="FetchListBox Example"
-- <string> Faith.FetchListBox(<string> name, <string> MenuName, <string> SubMenu)
Faith.FetchListBox("Spawn Vehicle", "veh", "Spawner")
```

### Faith.FetchListBoxIndex

This function will return the currently selected item's index id (starting from 0), useful when the string changes for example in a player list.

```jsx title="FetchListBoxIndex Example"
-- <int> Faith.FetchListBoxIndex(<string> name, <string> MenuName, <string> SubMenu)
Faith.FetchListBoxIndex("Spawn Vehicle", "veh", "Spawner")
```

## Examples

```jsx title="Basic MakeMenu with toggles on 2 submenus Example"
Faith.MakeMenu(ICON_FA_USER, "veh", "Spawner")
Faith.MakeMenu(ICON_FA_USER, "veh", "Tuning")
Faith.MakeToggle("Spawn Vehicle", "veh", "Spawner")
Faith.MakeToggle("Tune Vehicle", "veh", "Tuning") -- Put this into the Tuning submenu of Menu Vehicle

Faith.CreateWorker("Main loop", function()
  if Faith.FetchToggle("Spawn Vehicle", "veh", "Spawner") then
    -- Do something
  end

  if Faith.FetchToggle("Tune Vehicle", "veh", "Tuning") then
    -- Do something
  end
end)
```

```jsx title="Basic Menu Example"
--Initialize GUI and it's elements but only once
Faith.MakeWindow(ICON_USER_NINJA, "Local Menu", "Onself")
Faith.MakeToggle("GodMode", "Local", "Onself") 
Faith.MakeToggle("DemiGod", "Local", "Onself")
Faith.MakeToggle("High Jump", "Local", "Onself")
Faith.MakeToggle("Infinite Stamina", "Local", "Onself")
Faith.MakeToggle("Invisible", "Local", "Onself")
--Done with the GUI let's proceed to the features

Faith.CreateWorker("Main loop example menu", function()
  

    local p = PlayerPedId()
    local playerEntity = GetPlayerPed(p)
    local pID = PlayerId()
    
    local wasGodSet
    if Faith.FetchToggle("GodMode", "Local", "Onself") then
        SetEntityProofs(p, 1, 1, 1, 1, 1, 1, 1, 1)
      	wasGodSet = true
    elseif wasGodSet then
        SetEntityProofs(p, 0, 0, 0, 0, 0, 0, 0, 0)
      	wasGodSet = false --Avoid calling useless natives over and over
    end

    if Faith.FetchToggle("DemiGod", "Local", "Onself") then
        local hp = GetEntityHealth(p)
        if hp < 25 then
            SetEntityHealth(p, 100)
        end
    end

    if Faith.FetchToggle("High Jump", "Local", "Onself") then
        SetSuperJumpThisFrame(pID)
    end

    if Faith.FetchToggle("Infinite Stamina", "Local", "Onself") then
        RestorePlayerStamina(p, 100) -- I hope Default is 100, idfk.
    end
    local wasInvisSet
    if Faith.FetchToggle("Invisible", "Local", "Onself") then
        SetEntityVisible(p, 0, 0)
      	wasInvisSet = true
    elseif wasInvisSet then
        SetEntityVisible(p, 1, 0)
      	wasInvisSet = false --Avoid calling useless natives over and over
    end
end)
```
