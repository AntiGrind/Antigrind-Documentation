# VEHICLE_COMPONENT_CONTROL_EVENT

This event is called when an entity takes control of a vehicle's component like for example turrets.

```jsx title="Parameters"
-- Player handle / The player handle of the sending player.
-- Ped pedHandle / The handle of the ped.
-- bool isSeat / Whether the component is a seat.
-- bool permissionGranted / Whether permission was granted.


```
