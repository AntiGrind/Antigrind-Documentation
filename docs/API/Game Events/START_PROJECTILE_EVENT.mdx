# START_PROJECTILE_EVENT

This event is called when a player creates a projectile aka shooting something.

```jsx title="Parameters"
-- Player handle / The player handle of the sending player.
-- Entity target / The target entity.
-- Hash weaponHash / The hash of the weapon used.
-- Vector3 from / Where the trajectory started.
-- Vector3 to / Where it going.
-- bool singleShot / Whether it's a single shot or not.

```
