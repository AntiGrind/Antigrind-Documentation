# GIVE_WEAPON_EVENT

This event is called when a player gives a ped a weapon.

```jsx title="Parameters"
-- Player handle / The player handle of the sending player.
-- Ped pedHandle / The handle of the ped.
-- Hash weaponHash / The has of the weapon.
-- int ammo / How many bullets it gave.
-- bool isPickup / Whether it's a pickup or not.


```
