# OnModderDetection

### OnModderDetection
Gets called when a player is marked as a modder.

| Argument | Description                                          |
|----------|------------------------------------------------------|
| Handle   | The Handle of a player that can be used with natives |
| Name     | The Player's Name                                    |
| Reason   | Says why the player was marked as modder             |
